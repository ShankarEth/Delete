Text property

1. color                    -> color values (or) rgb(0-255) (or) hex #(0-9 a-f)
2. text-align               -> left, center, right, justify
3. text-decoration-line     -> underline, overline, line-through
4. text-decoration-color    -> color values (or) rgb(0-255) (or) hex #(0-9 a-f)
5. text-decoration-style    -> solid, dotted, dashed, wavy, double, none
6. text-decoration-thickness -> measurement in px, in, cm, %, em, rem
7. text-decoration          -> short hand property
8. text-indent              -> measurement in px, in, cm, %, em, rem
9. letter-spacing           -> measurement in px, in, cm, %, em, rem
10. word-spacing            -> measurement in px, in, cm, %, em, rem
11. text-shadow             -> x-axis y-axis blurness color [measurement measurement measurement color]
12. text-transform          -> uppercase, lowercase, captialize

Font property
1. font-size                -> measurement in px, in, cm, %, em, rem
2. font-weight              -> 100-900, bold, bolder, light, lighter
3. font-style               -> italic
4. font-variant             -> small-caps
5. font-family              -> Choose the font family

Background property

1. background-color         -> color values (or) rgb(0-255) (or) hex #(0-9 a-f)
2. background-image         -> linear-gradient() or radial-gradient() or url("imagePath")
3. background-repeat        -> repeat, no-repeat, repeat-x, repeat-y
4. background-position      -> top, bottom, center, left, right
5. background-attachment    -> fixed, scroll
6. background-size          -> cover

List property

1. list-style-type          -> disc, circle, square, none
                               decimal, alpha, roman
2. list-style-position      -> inside, outside
3. list-style-image         -> url("imagePath")
4. list-style               -> shorthand property

Table property

1. align                    -> left, right, center
2. v-align                  -> top, bottom, middle
3. caption-side             -> top, bottom
4. overflow-x               -> scroll, auto
